# HarpyNet-Router

Public installer and release packages for HarpyNet.

This repository intentionally contains only the router-facing installer and release assets. Source code is built and published from the main HarpyNet repository.

## Install

```sh
sh <(wget -O - https://raw.githubusercontent.com/sentiox/HarpyNet-Router/main/install.sh)
```

Packages are published in GitHub Releases as `.ipk` and `.apk` files.
